# DrawIO 配色方案

根据业务场景选择合适的配色方案。**默认使用「经典蓝橙」**，可根据需求切换。

所有节点/连线的颜色必须从下表中取值，不要手写颜色。弱化元素禁止使用 `opacity`，必须使用表格中"弱化元素"的颜色，并配合更细边框或虚线（`dashed=1;dashPattern=8 4`）实现层次。
